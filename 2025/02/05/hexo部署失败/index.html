查看密钥，并复制密钥

将 SSH 密钥添加到 GitHub

  1. 登录 GitHub，点击右上角头像 → Settings。
  2. 在左侧菜单中点击 SSH and GPG keys。
  3. 点击 New SSH key：
    • Title：填写一个描述（例如 My Laptop）。
    • Key：粘贴刚才复制的公钥内容。
  4. 点击 Add SSH key 保存。

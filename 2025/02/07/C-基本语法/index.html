C 和 C++ 的基本区别

C++ 是 C 的扩展，增加了许多新特性。

C 是面向过程的语言，适合底层开发；C++ 是 C 的超集，支持面向对象编程，功能更强大。

面向过程开发

首先，什么是面向过程：

把程序看成一系列步骤，一步步去执行。

就像做菜，按照食谱一步一步来：洗菜 → 切菜 → 炒菜 → 装盘。

它的程序由函数组成的，其中数据（变量）和操作（函数）是分开的，适合简单、线性的任务。

面向对象开发

C++ 的风格是面向对象编程，把程序看作一组对象，对象有自己的属性和行为。就像开餐厅，餐厅里有厨师、服务员、顾客等对象。每个对象都有自己的职责：

  • 厨师：负责做菜。
  • 服务员：负责上菜。
  • 顾客：负责点菜和吃饭。

其实有点像 python 中一个个类

它的程序由类和对象组成，数据（属性）和操作（方法）封装在一起，适合复杂的系统，易于扩展和维护。

C++ 的面向对象特性

C++ 支持面向对象编程，主要有以下特性：

类（Class）

  • 类是对象的模板，定义了对象的属性和行为。
  • 比如，Dog 类可以有属性 name 和方法 bark()。

封装（Encapsulation）

  • 把数据和方法封装在一起，隐藏内部细节。
  • 比如，你不需要知道手机内部如何工作，只需要知道怎么用。

继承（Inheritance）

  • 一个类可以继承另一个类的特性。
  • 比如，Cat 和 Dog 都可以继承 Animal 类的特性。

多态（Polymorphism）

  • 同一个方法在不同对象中有不同的行为。
  • 比如，Animal 类的 speak() 方法，Dog 会“汪汪”，Cat 会“喵喵”。

总结

  • C：面向过程，适合底层开发，代码简单直接。
  • C++：支持面向对象，适合大型项目，代码更易扩展和维护。
  • 面向过程：关注步骤和流程。
  • 面向对象：关注对象和它们之间的关系。
